# Agent Intelligence Template

This template contains the "Soul" and "Workflows" used for advanced hotel scraping and automated auditing.

## Contents
- **.agent/**: Workflows (git-push, automate-audit) and Skills.
- **SOUL.md**: The behavioral rules for the agent.
- **PROJECT_README.md**: The reference README for the hotel scraper.

## How to use in a new project:
1. Copy the `.agent/` folder and `SOUL.md` into your new project root.
2. Tell the agent: "Please read the .agent/ folder and SOUL.md to understand how I want you to work."
3. Your agent will now follow the mandatory testing and verbose logging rules automatically.
